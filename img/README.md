# img/

This directory is reserved for project images used in the README and
documentation.

Suggested contents :
- `architecture-diagram.png` — exported version of the architecture diagram
- `ptl-screenshot.png` — screenshot of PTL verbose output in a terminal
- `blender-example.png` — a sample render produced by the Blender bridge
- `claude-desktop-setup.png` — screenshot of the Claude Desktop config

Images should be optimized for web (under 500KB each). PNG for screenshots,
JPEG for renders.
